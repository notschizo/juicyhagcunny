---
title: Custom Redirect
description: Configure FxEmbed to redirect you to an alternative frontend such as Horizon or Nitter.
---

You can configure FxEmbed to redirect you to an alternative frontend such as Horizon or Nitter:

```
https://fxtwitter.com/set_base_redirect?url=https://xcancel.com
```

This sets a cookie on the domain you configure it on. If you set it on `fxtwitter.com`, it will not automatically apply to `fixupx.com` — you would need to set it separately on each domain.
