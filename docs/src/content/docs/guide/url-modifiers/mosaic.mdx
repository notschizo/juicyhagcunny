---
title: Mosaic
description: Combine multiple images from one post into a single composite with the m. prefix.
---

Combines multiple images from a single post into one composite image.

```
https://m.fxtwitter.com/user/status/123456789
https://m.fixupx.com/user/status/123456789
https://m.fxbsky.app/profile/user.bsky.social/post/abc123
```

On the majority of platforms other than Discord, mosaic is the default behavior.
