---
title: Translate
description: Append a language code to show translated post text
---

Append a two-letter ISO 639-1 language code to translate the post

```
https://fxtwitter.com/user/status/123456789/en
https://fixupx.com/user/status/123456789/es
https://fxbsky.app/profile/user.bsky.social/post/abc123/en
```

The embed will display both the original text and the translation.
