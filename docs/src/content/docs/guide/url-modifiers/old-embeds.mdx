---
title: Old embeds (Discord)
description: Use the o. prefix for the legacy Discord embed style when rich embeds misbehave.
---

Forces the old embed style for Discord. This can be useful for older Discord clients where the rich embeds might break videos.

```
https://o.fxtwitter.com/user/status/123456789
https://o.fixupx.com/user/status/123456789
https://o.fxbsky.app/profile/user.bsky.social/post/abc123
```
