---
title: Text-only
description: Show post text without photos or videos using the t. prefix.
---

The opposite of gallery, shows only the post text and excludes all photos and videos.

```
https://t.fxtwitter.com/user/status/123456789
https://t.fixupx.com/user/status/123456789
https://t.fxbsky.app/profile/user/post/abc123
```
