---
title: Select media
description: Target a specific image or video in a multi-media post with /photo or /video paths.
---

You can target a specific image or video from a multi-media post to only embed one:

```
https://fxtwitter.com/user/status/123456789/photo/3
https://fixupx.com/user/status/123456789/video/2
```

You can stack this with the [direct media](/guide/url-modifiers/direct-media/) flag to link directly to a specific image or video:

```
https://d.fxtwitter.com/user/status/123456789/photo/3
https://d.fixupx.com/user/status/123456789/video/2
```
