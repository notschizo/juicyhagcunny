---
title: Instant View
description: Force Telegram Instant View for a post or thread with the i. prefix.
---

Forces Telegram Instant View rendering for the linked post or thread.

```
https://i.fxtwitter.com/user/status/123456789
https://i.fixupx.com/user/status/123456789
https://i.fxbsky.app/profile/user.bsky.social/post/abc123
```
